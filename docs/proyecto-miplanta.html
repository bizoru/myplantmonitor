¿Qué es Miplanta?

Miplanta es un sistema completo de Internet de las Cosas (IoT) que monitorea una planta real en tiempo real. Los sensores miden temperatura, humedad del aire y humedad del suelo. Esos datos viajan por internet hasta una pantalla que cualquiera puede ver desde el navegador.

El sistema también permite controlar la luz de crecimiento LED de la planta desde cualquier lugar del mundo con un clic.

💡 Analogía simple: es como un termostato inteligente, pero para una planta. La planta "habla" con la nube cada 60 segundos y la nube guarda todo el historial.

Protocolo MQTT — Los "Temas" del Sistema

Tema (Topic)DirecciónContenido
sensors/plant-01/state ESP32 → Nube {"temperature":22.5,"humidity":61,"soil_moisture":75}
sensors/sim-web/state Simulador → Nube Mismo formato que el ESP32 pero desde el navegador
cmd/plant-01/light App → ESP32 {"on":true} o {"on":false}
state/plant-01/light ESP32 → App Confirmación del estado de la luz

Guía de Demostración (3 minutos)

🎯 Consejo: Practica esta secuencia al menos dos veces antes de la presentación. Abre las 3 pestañas con anticipación para no perder tiempo buscando URLs.

Parte 1 — Los datos en vivo (1 minuto)

  1. 1
    Abre https://miplanta.app/ en el navegador
  2. 2
    Señala las gráficas de temperatura, humedad y humedad del suelo
  3. 3
    Di: "Estos datos vienen del sensor conectado a la planta real, cada 60 segundos. InfluxDB los guarda durante 30 días para poder ver el historial."

Parte 2 — El simulador 3D (1 minuto)

  1. 1
    Abre https://miplanta.app/sim/
  2. 2
    Mueve los sliders de temperatura o humedad → los valores cambian en tiempo real en la pantalla 3D
  3. 3
    Presiona el botón PUBLICAR
  4. 4
    Di: "El dato acaba de viajar desde este navegador, pasó por internet, llegó al servidor en Virginia (AWS), y en unos segundos aparece en Grafana."
  5. 5
    Vuelve a Grafana y muestra el nuevo punto en la gráfica

Parte 3 — Controlar la luz (1 minuto)

  1. 1
    En el simulador, activa el interruptor de LUZ
  2. 2
    La tira LED de la planta 3D se enciende con luz rosada/morada (como los LEDs de cultivo reales)
  3. 3
    Di: "Este comando viajó del navegador al servidor, y de ahí al microcontrolador ESP32 que está físicamente conectado a la planta. Todo en menos de 1 segundo."

Preguntas Frecuentes de la Profesora

PreguntaRespuesta sugerida
¿Qué es IoT? Internet de las Cosas: objetos físicos conectados a internet que envían datos. Igual que un termostato inteligente o un reloj deportivo.
¿Por qué MQTT y no HTTP? MQTT usa muy poca batería y ancho de banda — perfecto para sensores pequeños que envían datos frecuentes. HTTP está pensado para páginas web.
¿Dónde están los datos? En un servidor AWS en Virginia, EE.UU. Base de datos InfluxDB, especializada en datos que cambian con el tiempo (temperaturas, sensores).
¿Qué aprendiste tú? "Aprendí a usar Node-RED para crear flujos de datos sin escribir mucho código. Entendí cómo los sensores se comunican por internet y cómo visualizar datos en tiempo real."
¿Cómo investigaste para este proyecto? "Investigué sobre IoT, MQTT y servidores en la nube. Armé los flujos en Node-RED y preparé la presentación."

Glosario Rápido

IoT
Objetos físicos conectados a internet que envían y reciben datos
MQTT
Protocolo de mensajes para máquinas — como WhatsApp pero para sensores
Broker
El servidor que recibe y distribuye mensajes MQTT (Mosquitto)
Topic / Tema
El "canal" de un mensaje MQTT, ej. sensors/plant-01/state
Node-RED
Programa visual para conectar sensores con bases de datos, sin código complejo
InfluxDB
Base de datos especializada en datos que cambian con el tiempo
Grafana
Programa para crear gráficas bonitas de datos en tiempo real
ESP32
Microcontrolador pequeño con WiFi que conecta sensores al internet
DHT22
Sensor de temperatura y humedad del aire
API
Interfaz para que programas se comuniquen entre sí por internet
HTTPS
Versión segura de HTTP (el candado verde en el navegador)
Docker
Sistema que empaqueta programas para que funcionen igual en cualquier servidor
AWS
Amazon Web Services — los servidores en la nube de Amazon
Time-series DB
Base de datos optimizada para guardar mediciones ordenadas por tiempo
WebSocket (WSS)
Conexión de internet bidireccional en tiempo real, usada por el simulador
Caddy
Servidor web que protege el sitio con HTTPS automático
